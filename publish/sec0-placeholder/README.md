# sec0

`sec0` is a minimal placeholder package published to reserve the npm package name.

## Install

```bash
npm install sec0
```

## Usage

```js
const sec0 = require("sec0");

console.log(sec0);
```

Output:

```json
{
  "name": "sec0",
  "reserved": true,
  "message": "The sec0 package name is reserved. A fuller public package will be published here later."
}
```

## Notes

This folder is publish-only and is intentionally not part of the workspace package graph.
